# 🚗 Car Price Prediction using Machine Learning

## 📌 Overview
This project aims to predict the **resale value of used cars** based on various specifications such as manufacturing year, mileage, fuel type, and engine capacity.  
The dataset used is sourced from [Kaggle – Car details v3](https://www.kaggle.com/datasets/nehalbirla/vehicle-dataset-from-cardekho), containing **8,000+ car listings**.

---

## 🎯 Objectives
- Perform **Exploratory Data Analysis (EDA)** to identify trends, outliers, and correlations.
- **Clean and preprocess** the dataset (handle missing values, convert strings to numeric).
- Apply **feature selection** techniques to identify important predictors.
- Train and evaluate regression models including **Linear Regression** and **Random Forest**.
- Measure performance using **R² Score** and **RMSE**.
